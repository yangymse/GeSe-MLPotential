The ML-learning potential is base on the AGNI code

== Installation of the HMLP package ==

1.  Please copy all the c++ code files in src to LAMMPS/src directory

2.  Compile LAMMPS (you should know how that works)

== Run tests ==

Run the example which includes test input file and gese.e2 file.
./lmp_serial < example/relax.in
